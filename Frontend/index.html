Documentation & Core Architecture

Welcome to the CutCuration pipeline layout. This studio optimizes multi-asset video raw inputs through background processing systems down to structured generative model payloads.

Preprocessing Node

Videos dropped inside RawVideos/ are processed using FFmpeg. To protect background task limits, files are down-scaled to 480p proxy targets at 1 FPS.

Concurrency Throttling

Active batch queues run safely underneath a semaphore token locking scheme (Cap: 2 concurrent tasks) to balance speed against rate limits.
