================================================================================
KEY RELATIONSHIPS & DATA FLOW
================================================================================

1. MOCK MODE TOGGLE
   Frontend: app.js (setMockMode) 
   → Backend: main.py (stream_pipeline) receives mock_mode=true/false
   → Uses services.py or config.py mock data
   → Output: report.csv and summary.txt

2. VIDEO PROCESSING PIPELINE
   Frontend: Click "Kickoff" button
   → Opens EventSource to /api/process/stream
   → main.py: stream_pipeline() generator
   → Calls services.py: process_single_video_with_logs() for each video
   → FFmpeg compression via preprocess.py
   → Gemini upload & analysis
   → CSV output to Backend/report.csv
   → Gemini blueprint to Backend/summary.txt
   → Frontend: Renders results

3. STRATEGY SELECTION
   Frontend: Select dropdown (id="sessionMode")
   → Value: "short_form_cooking" or "long_form_vlog"
   → Used by: config.py EDITING_CONFIGS[mode]
   → Passes: Specific prompts to Gemini

4. FILE STRUCTURE FOR EDITING
   LAYOUT CHANGES: Frontend/index.html + Frontend/css/styles.css
   STYLE CHANGES: Frontend/css/styles.css
   LOGIC CHANGES: Frontend/js/app.js
   API ENDPOINTS: Backend/App/main.py
   PROCESSING LOGIC: Backend/App/services.py
   PROMPTS: Backend/App/prompts.py
   CONFIGURATION: Backend/App/config.py


================================================================================
COMMON EDITING SCENARIOS
================================================================================

SCENARIO 1: Add a new editing strategy
   1. Edit Backend/App/prompts.py - Add new EDITING_CONFIGS entry
   2. Edit Frontend/js/app.js - Add to CLIENT_PROMPT_MAP
   3. Edit Frontend/index.html - Add option to select element

SCENARIO 2: Change video compression quality
   1. Edit Backend/App/preprocess.py - Modify ffmpeg_cmd parameters
   2. Edit Backend/App/services.py - Same changes to preprocess_video_for_tokens

SCENARIO 3: Modify Gemini prompts
   1. Edit Backend/App/prompts.py - Update analysis_prompt or summary_prompt
   2. Test with mock mode ON (Backend/App/config.py: MOCK_DEV_MODE = True)

SCENARIO 4: Change UI layout
   1. Edit Frontend/index.html - Modify structure
   2. Edit Frontend/css/styles.css - Update styling
   3. Edit Frontend/js/app.js - Update selectors/event handlers

SCENARIO 5: Add new API endpoint
   1. Edit Backend/App/main.py - Add @app.get/post decorator
   2. Edit Frontend/js/app.js - Add fetch() call
   3. Test with mock mode OFF and real API (if needed)

SCENARIO 6: Change rate limiting
   1. Edit Backend/App/main.py - Modify concurrency_semaphore = asyncio.Semaphore(X)
   2. Edit Backend/App/services.py - Modify semaphore = asyncio.Semaphore(X)
